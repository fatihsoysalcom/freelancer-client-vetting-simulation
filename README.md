# Freelancer Client Vetting Simulation

This Python script simulates the critical aspect of client vetting in freelancing. It demonstrates how freelancers might prioritize or be more cautious with unverified clients due to increased perceived risk, impacting project acceptance and freelancer reputation.

## Language

`python`

## How to Run

Save the code as `freelancer_vetting.py` and run it from your terminal using: `python freelancer_vetting.py`
